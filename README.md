# PointMaster

Web app for managing offline card game scores (Twenty-Nine, Call Bridge).

## Stack

- **Frontend:** Next.js (App Router), HeroUI, Tailwind CSS
- **Backend:** Express.js, MongoDB, Socket.IO
- **Auth:** Better Auth
- **Language:** JavaScript

## Structure

```
pointmaster/
├── client/    # Next.js frontend
├── server/    # Express.js backend
└── docs/      # Specifications
```

## Getting Started

```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```
